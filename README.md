# tinymediamanager

An movies and tv shows organizer.

URL: http://www.tinymediamanager.org/

Optional from KCP:
[libzen](../../../libzen) [libmediainfo](../../../libmediainfo) [mediainfo](../../../mediainfo)

Install in this order:
```
kcp -di libzen 
```
```
kcp -di libmediainfo 
```
```
kcp -di mediainfo 
```
```
kcp -i tinymediamanager
```
